Project Overview

The Student Performance Predictor is an intelligent machine learning application designed to predict student academic performance based on key factors that influence learning outcomes.

How It Works

  • Study Hours: The amount of time dedicated to studying
  • Attendance: The percentage of classes attended by the student
  • Sleep Hours: The daily hours of sleep for proper cognitive function

The application uses a trained machine learning model to analyze these three critical factors and predict the expected marks a student will achieve. It also provides a clear PASS/FAIL status based on the predicted marks.
